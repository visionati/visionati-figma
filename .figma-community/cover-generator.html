The cover below is exactly 1920×1080 pixels. To export as PNG:

  1. Open this file in Chrome or Firefox
  2. Open DevTools → select the .cover element
  3. Right-click the element → Capture node screenshot (Chrome)
    or use :screenshot --selector .cover in Firefox console
  4. The saved PNG will be exactly 1920×1080 — replace cover-1920x1080.png

Tip: Make sure browser zoom is at 100% and device pixel ratio is 1× for exact dimensions.
On Retina/HiDPI displays, you may get 2× resolution (3840×2160) which Figma will accept and look sharper.
